# docs/archive

Este diretório contém documentos de análise, planos, relatórios e artefatos de tarefas gerados durante spikes, auditorias e migrações.

**Objetivo**: reduzir ruído no contexto de agents de IA (Copilot, Claude, etc.) sem perder o histórico.

Os arquivos aqui são mantidos no Git para referência humana e histórico, mas devem ser ignorados por padrão pelos agents através de:
- .copilotignore
- .claudeignore
- Configurações de workspace (search.exclude, files.exclude)

Não edite estes arquivos a menos que esteja revisando o histórico da decisão.
